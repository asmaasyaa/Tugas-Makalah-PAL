# Objek Kajian: Praktik Automasi Layanan

Repository ini berisi skrip praktik mandiri untuk mendukung Tugas Makalah Penyediaan dan Automasi Layanan.

## Daftar File
1. **`Dockerfile`**: Representasi dari praktik *Containerization* (Bab 3) untuk membungkus web server Nginx sederhana.
2. **`main.tf`**: Representasi dari praktik *Infrastructure as Code* (Bab 5) menggunakan Terraform untuk mendeskripsikan server EC2 di AWS.
